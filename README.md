# تطبيق التحضير للامتحان - مقياس تحليل البيانات

تطبيق ويب تفاعلي متطور مصمم لمساعدة الطلاب على التحضير للامتحانات النظرية في مادة **تحليل البيانات والإحصاء** من خلال نظام QCM (أسئلة متعددة الخيارات).

## 🚀 ميزات التطبيق

- **تعدد مستويات الصعوبة:** (سهل، متوسط، صعب) مع 10 أسئلة لكل مستوى.
- **تفسير إحصائي فورى:** عند الإجابة، يظهر شرح علمي يوضح "لماذا" هذه هي الإجابة الصحيحة لتعزيز الفهم وليس الحفظ فقط.
- **خلط الخيارات ذكياً:** يتم تغيير ترتيب الخيارات عشوائياً في كل محاولة لمنع الحفظ البصري لمكان الإجابة.
- **تتبع التقدم بيانياً:** سجل كامل لنتائج الاختبارات السابقة مع رسم بياني يوضح تطور مستواك عبر الزمن.
- **نظام مؤقت مزدوج:** 
  - مؤقت كلي للامتحان (15 دقيقة) لمحاكاة ضغط الامتحان الحقيقي.
  - مؤقت لكل سؤال (30 ثانية) لزيادة سرعة البديهة.
- **تجربة مستخدم احترافية:** 
  - يدعم **الوضع الليلي (Dark Mode)** لإراحة العين أثناء الدراسة.
  - مؤثرات صوتية تفاعلية تعطي تغذية راجعة فورية.
  - تصميم عصري وبسيط (Clean Minimalism).

## 🛠 التكنولوجيا المستخدمة

- **Frontend:** React.js / TypeScript.
- **Styling:** Tailwind CSS / UI Modern Design.
- **State Management:** React Hooks.
- **Charts:** Chart.js لتصوير البيانات الإحصائية للطالب.
- **Animations:** Motion for smooth transitions.

## 📦 النشر والتشغيل

التطبيق جاهز للنشر باستخدام Docker، ومتوافق تماماً مع منصة **Coolify**.

### التشغيل محلياً باستخدام Docker:
```bash
docker build -t qcm-analyzer .
docker run -p 80:80 qcm-analyzer
```
